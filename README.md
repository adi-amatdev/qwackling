# Repository Layout

This workspace contains the Duckling Python wrapper in:

- `duckling-wrapper`

That directory is the single source of truth for:

- package code
- tests
- build commands
- integration instructions
- PyPI publishing steps

Start there:

```bash
cd duckling-wrapper
```
